📜 Aviso de tratamiento de datos personales
En cumplimiento de la Ley 1581 de 2012 y el Decreto 1377 de 2013 sobre protección de datos personales en Colombia, el Conjunto El Mañana te informa que:

1. Responsable del tratamiento: Administración del conjunto residencial.
2. Finalidad: tus datos personales (nombre, documento, dirección dentro del conjunto, contacto, placas de vehículos) se usan únicamente para: generar tu carnet digital, identificarte en portería, tramitar tus solicitudes administrativas y enviarte comunicados oficiales.
3. Almacenamiento: la información queda guardada localmente en tu dispositivo. Solo se transmite cuando tú generas una solicitud o envías un comprobante, y exclusivamente a los correos oficiales del conjunto.
4. Derechos: puedes conocer, actualizar, rectificar o solicitar la supresión de tus datos en cualquier momento, escribiendo a la administración o usando la opción "Cerrar sesión y borrar datos" dentro de la app.
5. Seguridad: no compartimos tus datos con terceros diferentes a los órganos de administración (administrador, consejo, contador) del conjunto.
Aviso: Tus datos quedan guardados únicamente en este dispositivo. No se envían a ningún servidor. Solo se transmiten cuando tú generas una solicitud, y solo hacia los correos oficiales del conjunto.

🚀 Publicar a todos los residentes

Genera un archivo datos.json con la configuración actual y las noticias. Súbelo al repositorio GitHub (Add file → Upload files) reemplazando el anterior. Todos los residentes verán los cambios al abrir la app.

📝 Cómo subirlo a GitHub
  1. Toca "📦 Generar datos.json" arriba. Se descargará un archivo.
  2. En tu celular, abre tu repositorio en GitHub (app o navegador).
  3. Entra a elmanana-app-v2 → toca Add file → Upload files.
  4. Selecciona el archivo datos.json recién descargado.
  5. Si ya existe uno, GitHub preguntará si reemplazar: di que sí.
  6. Baja hasta "Commit changes" → Commit changes.
  7. En 1–2 minutos todos los residentes verán los cambios al abrir la app.
